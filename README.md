Crypto Currencies Analysis Project

This project analyzes different **Cryptocurrency markets such as Bitcoin and Ethereum using data analysis and visualization techniques. The goal is to study price trends, 
market capitalization, and trading volume to understand market behavior. Using tools like Python and libraries such as Pandas and Matplotlib,

project - 2
Global Education & Economy Dashboard

This project analyzes the relationship between education indicators and economic growth across different countries.
Using datasets on literacy rate, school enrollment, GDP, and population, the project performs exploratory data analysis and visualizations to identify patterns and trends.
An interactive dashboard was developed using Streamlit with data processing in Python using libraries like Pandas and Matplotlib.
The dashboard allows users to explore country profiles, run SQL queries, and visualize the impact of education on economic development.


project - 3
Aircraft Flight Ticket Price Prediction

This project predicts the price of airline tickets based on different factors such as airline company, source and destination cities, departure time, duration of the flight, 
and number of stops. Using historical flight data, machine learning models are trained to analyze patterns and estimate ticket prices. Data preprocessing, feature encoding, 
and exploratory data analysis were performed using Python with libraries like Pandas and NumPy. 
Prediction models such as Random Forest and Linear Regression were used to improve accuracy. The model helps users estimate the expected flight ticket price before booking.
